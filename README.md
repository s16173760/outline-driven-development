# Outline-Driven Development

This repository contains notes and examples for outline-driven development.

## Install

### Windows PowerShell Install

```powershell
cargo install ast-grep ripgrep fd-find eza lsd tokei bat git-delta just raff-cli tree-sitter-cli difftastic jj-cli
cargo install procs fselect tealdeer srgn nomino shellharden grex mergiraf jql huniq lemmeknow hyperfine rargs eva fend
cargo install numbat-cli hck

cargo install --locked git-branchless
cargo install --locked zoxide
cargo install nu
cargo install --locked sccache
cargo install --locked lazyjj
```
